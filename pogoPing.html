pogoPing

Description

pogoPing creates log files while playing online games in direct connection modus (e.g. PES, FIFA). In case, one of the players has complaints about the other one intentionally creating a “laggy” connection, those log files might resolve the situation.

Private keys Only pre-approved people will receive private decryption keys. Contact me if you are a league admin or would like to have your own set of keys.
